Tutorials

Hands-on walkthroughs of Morph

Three short tutorials that take you from an empty directory to a fully tracked, eval-gated project — covering the parts of Morph that don't exist in Git: runs, traces, behavioral commits, and merge by dominance.

01

Start a new project with Morph

From morph init through metric-bearing commits, an agent session recorded as a Trace, and a clean revert — all without Git in the picture. The fastest way to feel the whole loop.

~20 min Morph only Python · pytest Intro
02

Add Morph to a project that already has Git

Layer Morph onto a real Git repository in reference mode. Git stays the source of truth, every git commit is automatically tied to a Morph commit by hash, and your team's workflow is unchanged — they don't need Morph installed at all.

~20 min Reference mode Python · pytest · Git Intermediate
03

Branches, merges, and merge conflicts

Two branches, two metric vectors, one behavioral merge gate. Walk through dominance, the union eval suite, case provenance, and what to do when the metrics genuinely conflict.
